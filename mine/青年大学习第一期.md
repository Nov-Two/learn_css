### 如何编写实习经历 & 包装简历

不贩卖焦虑，也不假大空，我们直接开始吧。

除了个人信息，校园经历，如何虚构一段不存在的经历。你现在想找一份工作，缺少经验，不知道自己要做什么，或者你对上一份实习经历并不满意，想要换一个新的环境，我分3部分说。

#### 包装在校经历

1. 核心原理：角色互换，重新审视自己的在校经历，心里不要在去想我在学校学了什么，完成了什么任务，转变思想与身份，把你的在校经历当成真实的职场环境，领导分发了什么任务，是独自解决，还是跨部门解决，解决这个问题之后得到了什么结果
2. 实操：
   - 在校老师让你完成一个学业汇报PPT，那你就角色互换说自己在公司整理项目方案PPT，这个PPT的内容你就可以说（项目背景，项目需求，面向群体，解决什么问题等等）
   - 怎么完成这个PPT的？你可以说自己调研了市场，调研了竞品公司，收集了哪些数据，网上资料查阅了哪些。
   - 结果是什么？这个PPT是否符合公司的需求，是否被采纳，哪里遇到了问题，学校有没有其他同学的帮助（公司有没有其他同事的帮助），获得了怎样的肯定
   - 完成这个学到了什么？收集信息，同学互助（同事协助），人员沟通
3. 举例
   | 项目 | 在校经历 | 职场环境 |
   | :--- | :--- | :--- |
   | 课程项目 | 帮助课题组收集材料 | 与公司同事合作，协调项目进度，解决项目资料问题。 |
   | 奶茶店兼职 | 在奶茶店摇奶茶，打单打包 | 独自完成工作，与顾客交流，处理订单，打包商品。 |
   | 个人项目 | 完成毕设 | 独立完成项目，与老师（公司）沟通，确定项目需求，与团队合作，解决项目问题。 |
   | 校内卖流量卡 | 负责在学校卖流量卡，与学生沟通，确定需求，与学生合作，解决问题。 | 通过线下或线上推销，完成了多少的指标，学到了哪些与人沟通谈判的技巧。 |
   | 拍视频 | 给同学拍照摄影 | 拍摄素材，给公司拍宣传，剪视频 |
4. 就是把你在笑的经历包装一下，假大空一下，用一些行业专业的词语装饰一下，有点类似行业黑话，尽管有的面试官不喜欢看这种简历

#### 虚构实习经历

1. 核心原理：把不真实的变为真实，注意几个方面，可以吹牛，但不能不符实际，要保证拥有细节和逻辑。
2. 实操：幻想自己上一份工作是你喜欢的事情，或者家里有没有人做生意之类的，把自己代入。在这里面你给自己一个角色，一个定位，你可以上网查查这个岗位的要求，需要掌握哪些技能，然后根据自己的情况，去完成这个岗位的任务。如果要在实习简历上体现的话，你就简单学习了解一下，不要一问三不知。
3. 举例：
   | 职业 | 幻想 |
   | :--- | :--- |
   | 外贸 | 找客户、报价、跟单、处理合同和物流文件、邮件沟通 |
   | 文员 | 录入数据、整理文档、安排会议、接待、跑腿杂务 |
   | 程序员 | 写代码、改bug、看需求文档、代码review、部署上线 |
   | 销售 | 打电话、拜访客户、发方案、谈价格、签合同、维护客情 |
   | 运营 | 写文案、排版发帖、看数据、策划活动、回复评论私信 |
   | 行政 | 订机票酒店、管办公用品、办入离职手续、组织团建、打杂 |
   | 财务 | 做账、报销审核、开发票、对账、月底出报表、配合审计 |
